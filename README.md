## 소개 문구

하루 두 번, 오전 11시, 오후 6시.

학과의 새 소식을 메일로 전달해드립니다:)

저는 부산대학교 전기컴퓨터공학부 정보컴퓨터공학전공 18학번
김민석이라고 합니다. 평소 학과 홈페이지의 게시판을 자주 뒤졌었는데
이게 너무 불편해서 직접 개발했습니다. 타과생, 타학교 학생들도 쓸 수
있게끔 설계하였습니다. 디자인은 갖다버리고 기능에만 집중하였습니다. 이
서비스가 도움이 되었으면 좋겠습니다:) 많이 쓰일수록 포폴로 쓰기
좋거든요.
